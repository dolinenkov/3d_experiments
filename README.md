# xrndr

Simple realtime rendering engine written for educational purposes

## Dependencies:

- glew
- SDL2
- glm
- fmt
- SOIL
- Assimp

## Controls

W,A,S,D + mouse - moving camera
F1 - toggle wireframe/filled polygon mode
Esc - exit
